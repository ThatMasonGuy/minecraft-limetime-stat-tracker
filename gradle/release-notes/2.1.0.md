# Lifetime Stat Tracker 2.1.0

Initial Modrinth release for Minecraft 1.21.11.

- Tracks lifetime stat totals across worlds, servers, Realms, and deleted saves.
- Adds per-world and per-server breakdowns for play time, distance, jumps, deaths, mob kills, and advancement history.
- Supports client-only installs, with optional Fabric server install for better multiplayer world identity.
- Includes `/lst` and `/lifetimestats` client commands plus `/lstserver identity` for server-side identity debugging.
- Stores data under `config/lifetime-stat-tracker/` with backup-backed clear/remove operations.
