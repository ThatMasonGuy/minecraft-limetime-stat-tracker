# Lifetime Stat Tracker 2.8.0

Launcher-agnostic data storage for Minecraft `1.20` through `26.2-pre-3`.

Choose the file that matches your Minecraft range:

- `1.20-1.20.4`
- `1.20.5-1.21.10`
- `1.21.11`
- `26.1-26.2-pre-3`

Highlights:

- Lifetime Stat Tracker now writes saved JSON data to a fixed per-user app-data
  folder outside `.minecraft`, so different launchers and launcher instances can
  share one tracker history.
- On first run, existing data from the current launcher's
  `.minecraft/config/lifetime-stat-tracker/` folder is copied into the new
  app-data folder if the app-data folder does not already contain tracker data.
- Legacy launcher-local files are left untouched as a safety backup.
- If the app-data folder already contains tracker data, old launcher-local data
  is not imported automatically to avoid accidental duplicate counting.
